## v0.7.0 - 2026-06-16
### Added
* Slog spec-compliance tests ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Comprehensive benchmark suite ([#6](https://github.com/desertwitch/slog-seq/pull/6))
* Comprehensive documentation and comments ([#6](https://github.com/desertwitch/slog-seq/pull/6))
* `NewSeqHandler` as new primary entry-point to establish `slog.Handler` ([#9](https://github.com/desertwitch/slog-seq/pull/9))
* `WithEventEnricher` option to call custom event enriching before dispatch ([#9](https://github.com/desertwitch/slog-seq/pull/9))
* `WithBufferSize` and `WithRetryBufferSize` options to control queue sizes ([#10](https://github.com/desertwitch/slog-seq/pull/10))
* Allow zero value for `WithFlushInterval` to disable periodic flushing ([#10](https://github.com/desertwitch/slog-seq/pull/10))
### Changed
* Split OTel integration into separate `slog-seq/seqotel` package ([#9](https://github.com/desertwitch/slog-seq/pull/9))
### Fixed
* Close() teardown causing panics, blocked senders, and lost events ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Handle() mutating the caller's slog.Record ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* WithGroup/WithAttrs interleaving producing incorrect nesting ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Oversized batches being dropped instead of split and retried ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Missing HTTP client request timeout ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* HTTP response body not drained, breaking connection reuse ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Multiple WithGlobalAttrs calls overwriting instead of accumulating ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* retryBuffer silently lost on Close() ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* retryBuffer not flushed on periodic tick ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* User properties overriding reserved CLEF keys ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* ReplaceAttr receiving wrong group scope ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Zero worker count causing panic ([#1](https://github.com/desertwitch/slog-seq/pull/1))
* Zero/negative batch size and flush interval not validated ([#5](https://github.com/desertwitch/slog-seq/pull/5))
* ReplaceAttr and error conversion not applied to nested attrs ([#5](https://github.com/desertwitch/slog-seq/pull/5))
* WithGlobalAttrs not resolving LogValuer types eagerly ([#5](https://github.com/desertwitch/slog-seq/pull/5))
* Refactored attr/group handling for correctness and performance ([#5](https://github.com/desertwitch/slog-seq/pull/5))
